# 📂 File Organizer — Neo-Brutalist Smart Sorter

A lightweight, automated desktop utility built to eliminate folder clutter instantly. This project bridges a robust **Python automation backend** with a striking, highly customized **React frontend** built around the **Neo-Brutalism** design philosophy. 

Developed as part of **Task 5** during my software engineering internship at **Synent Technologies**.

---

## ✨ Features

### 🎨 Human-Centric UI (Neo-Brutalist Redesign)
Pivoting away from standard, generic dark-mode SaaS templates, this user interface embraces a distinct, tactile aesthetic:
* **High-Contrast Borders & Hard Shadows:** Uses thick black outlines and opaque, crisp block shadows for an tactile, comic/arcade feel.
* **Muted Pastel Blocks:** Leverages a custom, high-contrast warm beige background paired with soft lavender, mint green, and pale yellow category accents.
* **Interactive Previews:** Users can drag and drop or browse files to see a live, animated visual breakdown of how their filesystem will be restructured before committing to changes.

### ⚙️ Automation Backend
* **Python Watchdog Integration:** Features a real-time system script that actively monitors designated local directories.
* **Instant Automation:** The split second a new file is detected, it is analyzed and instantly sorted into categorized directories (Images, Documents, Code, Archives, etc.) based on its extension.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite), Custom CSS3 Architecture, Google Fonts (`Public Sans` & `Space Mono`)
* **Backend:** Python 3, `watchdog` library (File system events monitoring)
